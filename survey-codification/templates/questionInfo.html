Survey Codifier Metrics Question Types
# Type Description
1 Horizontal This tag is used to describe the X axis of a grid type question.
2 Vertical This tag is used to describe the Y axis of a grid type question.
3 Single Response Used to identify questions that require the user to only select one of the given answers. Ex. Please select your favorite genre of movies.
4 Multiple Response Used to identify questions that allow the user to select one or more of the of the given answers. Ex. How would you characterize this trailer? Please select all that apply.
5 Grid Response A question in which an horizontal and vertical axis is used, that way a range of responses can be recorded. Ex. How likely are you to take the following actions? Please select all, Extremely likely, Very likely, Somewhat likely, Not so likely, Not at all likely.
6 Open Ended This tag identifies questions that do not have a series of answers to select from instead the user may respond as they wish. Ex. What motivates you most to watch a movie?

Steps and Documentation

1) Download the template and modify the questions given at your discretion, taking care to keep the same format of the downloaded document. The given document is divided using heading styles, the question is of type "Heading 1" and the answers are of type "Normal". (Word default text)

2) Submit the modified word document, a JSON object is returned.

3) The object is structured with the question number as a key and the question type, metric type, answers and literal question are an array of attributes.

Parser Logic

The parser is a Python application that uses the Python-docx and lxml libraries. It utilizes tags and heading styles to differentiate each question and its respective answers. The tags function as a simple text protoccol so the parser can identify which portions of the text correspond with the information we would like to obtain.
